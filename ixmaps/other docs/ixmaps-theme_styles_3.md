## Usage Notes

- **Type**: Most properties accept string values that are converted to appropriate types internally
- **Boolean values**: Use `"true"` or `"false"` strings
- **Numeric values**: Use string representations of numbers
- **Color values**: Use hex color codes (e.g., `"#FF0000"`)
- **Arrays**: Use array notation for multi-value properties like `colorscheme`

## Common Style Combinations

### Choropleth Map
```javascript
.style({
    type: "CHOROPLETH",
    colorscheme: ["5", "#FFFDD8", "#B5284B"],
    fillopacity: "0.7",
    bordercolor: "#FFFFFF",
    borderwidth: "1"
})
```

### Bubble Chart
```javascript
.style({
    type: "BUBBLE",
    colorscheme: ["5", "#FFFDD8", "#B5284B"],
    fillopacity: "0.7",
    stroke: "true",
    strokecolor: "#000000",
    strokewidth: "1"
})
```

### Symbol Map
```javascript
.style({
    type: "SYMBOL",
    symbolsource: "icon.svg",
    symbolsize: "24",
    symbolopacity: "0.8"
})
```
